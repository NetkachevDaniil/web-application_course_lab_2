# ЛР2 Flask

В работе реализованы страницы для вывода параметров URL, заголовков, cookie и данных формы авторизации, а также отдельная POST-форма проверки телефона с серверной валидацией и выводом ошибок через `is-invalid` и `invalid-feedback`. При корректном вводе номер приводится к формату `8-***-***-**-**`.
